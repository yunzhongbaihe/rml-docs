### 检查你电脑上是否有SSH Key
- ~/.ssh
- 存在：`bash: /c/Users/.../.ssh: Is a directory`
- 不存在：`bash: /c/Users/.../.ssh: No such file or directory`
### 创建SSH Key
- ssh-keygen -t rsa -C "你的邮箱"

### 引入存在的文件作为代码片段
- `<<< @/文件夹名称/文件名称`
- @/：为当前项目所在的位置，绝对路径，例如：`E:\shop\`
